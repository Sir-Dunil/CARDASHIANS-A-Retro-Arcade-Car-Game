CARDASHIANS: A Retro Arcade Car Game
A custom-built, retro-style arcade game featuring real-time obstacle avoidance, integrated audio, and interactive hardware controls. This project was developed as a collaborative effort to showcase end-to-end embedded system integration.
🛠 Technologies Used
Hardware:
Microcontroller: Arduino UNO
Display: TFT Capacitive Touch Display (Main Game) & LCD (Scoreboard)
Audio: MP3 Player Module + 2GB SD Card + 1W, 8 Ohms Mini Speaker
Input Controls: Joystick (Vehicle Control) & Push Buttons (Start/Reset)
Software:
Language: C++
Development Environment: Arduino IDE
📋 Project Overview
CARDASHIANS was developed to create a functional, immersive arcade experience using limited embedded resources. The project required precise timing for game logic, multi-display management, and serial audio playback, effectively demonstrating the integration of disparate hardware modules into a unified, responsive system.
🚀 Key Features
Dual-Display Architecture: Utilizes a TFT capacitive display for primary gameplay rendering and a dedicated LCD for clear, real-time tracking of current scores and high scores.
Immersive Audio Engine: Integrated MP3 player module and dedicated speaker system to deliver game-specific sound effects and ambient audio, stored via SD card.
Intuitive Control Interface: Features a responsive joystick input for fluid vehicle navigation, complemented by hardware push buttons for seamless game state management (Start/Reset).
Embedded Logic Integration: Leverages the Arduino UNO to manage game state, collision detection, and concurrent hardware communication between the display, audio module, and input peripherals.
